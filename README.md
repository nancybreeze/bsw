# BSW Website

Static website for the **Cynanchum auriculatum Research Center (白首乌研究中心)**.

## Purpose
- Present basic information on *Cynanchum auriculatum*  
- Summarize research directions (genomics, pharmacology, plant resources)  
- Provide links to public datasets and published results  

## Deployment
Hosted via **GitHub Pages**:  
https://nancybreeze.github.io/bsw/

- `index.html` — main page  
- `style.css` — styling
